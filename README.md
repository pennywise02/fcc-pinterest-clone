# Free Code Camp Pinterest Clone
Ecommerce app Using the MongoDB driver and Express

# Technology Stack
- Mongo
- Express.js
- Node.js
- Bootstrap
- jQuery
- Masonry Grid

## Installation
```bash
git clone https://github.com/ratracegrad/fcc-pinterest-clone
npm install
bower install
node server.js
```
